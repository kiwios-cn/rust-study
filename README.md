# rust-study

Rust 语言学习笔记与练习。

## 目录结构

```
rust-study/
├── guessing_game/     # 猜数字游戏（Rust 入门经典）
├── varible/           # 变量、所有权、借用
└── data_type/         # 数据类型、枚举、结构体
```

## 主要内容

- **所有权系统** — ownership、borrowing、lifetime
- **基础类型** — 标量类型、复合类型、枚举
- **控制流** — if/loop/while/for
- **函数与闭包** — fn、closure、迭代器
- **错误处理** — `Result`、`Option`、`?` 运算符

## 运行

```bash
cd guessing_game
cargo run
```
